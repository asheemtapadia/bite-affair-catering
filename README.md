# Bite Affair Catering Website

This is the official website project for Bite Affair Catering.

---

## Project Overview

This project is a modern catering website built to showcase services, menu, and allow users to explore offerings easily.

---

## Tech Stack

- Vite
- TypeScript
- React
- Tailwind CSS
- shadcn-ui

---

## How to Run Locally

Follow these steps to run the project on your system:

```sh
git clone https://github.com/asheemtapadia/bite-affair-catering.git

cd bite-affair-catering

npm install

npm run dev
